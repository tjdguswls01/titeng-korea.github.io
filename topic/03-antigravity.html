03 / Antigravity AI IDE

Antigravity IDE 활용 흐름

GitHub에서 관리되는 코드와 문서를 AI IDE가 어떻게 탐색하고, 기획 메모에서 프로토타입까지 연결하는지 발표 흐름으로 정리합니다.

GitHub Antigravity 참고 영상

AI IDE loop

Plan요구사항과 발표 목표를 작은 작업 단위로 나눕니다.
Explore폴더 구조, 링크, 기존 코드 패턴을 빠르게 확인합니다.
Build프로토타입, HTML 섹션, 문구 초안을 생성합니다.
Review변경 범위와 링크 오류를 검증합니다.

Code 탐색

기존 티아이티 이엔지 기술 포털과 발표 자료 구조를 파악합니다.

Prototype

페이지 초안, 설명 문구, 데모 흐름을 빠르게 생성합니다.

검토

주제와 맞지 않는 요소를 덜어내고 링크와 표현을 통일합니다.

발표 메모

  • AI IDE는 결과물을 바로 신뢰하기보다 탐색 속도를 올리고 초안을 빠르게 만드는 도구로 설명합니다.
  • 기존 디자인 패턴을 가져오되, 발표 주제와 맞지 않는 기능은 제외하는 기준을 보여주기 좋습니다.
  • Canva 자료처럼 아직 없는 콘텐츠는 placeholder를 먼저 만들고 나중에 교체하는 흐름을 소개합니다.
