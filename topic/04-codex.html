04 / Codex CLI with VS Code

Codex CLI with VS Code

발표 자료처럼 여러 HTML 파일과 링크가 얽힌 작업을 Codex CLI로 수정하고, VS Code에서 검토 가능한 상태로 만드는 흐름을 다룹니다.

Antigravity Codex 참고 영상
codex-workflow.md

$ codex inspect presentation.html

$ codex update topic pages

$ codex verify links

✓ review-ready static deck

무엇을 도와주나

폴더 구조 생성, HTML 뼈대 정리, 링크 수정, 반복 페이지 표기 통일, 간단한 검증 명령 실행을 도와줍니다.

발표 포인트

사람이 검토할 수 있는 변경 단위로 작업을 나누고, 결과물을 파일과 링크 기준으로 확인하는 과정이 핵심입니다.

1. 요청 정리

주제, 대상 파일, 유지할 디자인 요소와 제외할 기능을 명확히 합니다.

2. 파일 수정

기존 패턴을 존중하면서 발표 자료에 필요한 섹션만 구현합니다.

3. 검증

내부 링크, 상대 경로, 표기 통일, 브라우저 확인 가능성을 점검합니다.

발표 마무리

Product에서 시작해 GitHub, Antigravity, Codex로 이어지는 실제 작업 흐름을 한 번에 회고합니다.
