AI Tool Evolution

AI 코딩 도구의 진화: Copilot → Agent

🖊
Stage 1 · 2022
줄 단위 자동완성 (Copilot)
다음 줄 코드를 확률적으로 제안. 탭 키로 수락.
💬
Stage 2 · 2023
대화형 챗봇 통합 (Chat Mode)
자연어로 질문→코드 블록 생성. 파일 단위 수정 가능.
🤖
Stage 3 · 2024~ 현재 주류
자율형 에이전트 (Cursor / Windsurf)
프로젝트 전체 구조 자동 파악 → 멀티 파일 동시 수정 → 자율적 테스트 실행 및 오류 수정.
⚠️ Risk Factor

AI가 발생시키는 위험 요소: 환각(Hallucination)

실제 발생 사례 패턴
// AI가 생성한 코드 (실행 시 오류)
import { smartFilter } from 'lodash-extended';
// ❌ 존재하지 않는 패키지

userArray.groupByNestedKey('dept.id');
// ❌ 실제로 없는 Array 메서드
환각이 발생하는 이유

LLM은 확률적 패턴 생성기이므로, 존재하지 않는 API도 그럴듯하게 조합해 출력합니다. 특히 최신 라이브러리나 마이너 패키지일수록 위험도가 높습니다.
