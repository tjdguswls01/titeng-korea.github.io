tPay NFC 모바일 SDK를
10분 만에 연동하는 방법

복잡한 카드 터치 결제 연동 과정은 이제 그만. iOS/Android 하이브리드 가이드부터 POS 하드웨어 인터페이스 직렬화 제어까지 실제 동작 코드를 통해 가장 안전한 모바일 핀테크 인프라를 구축해 보세요.

SDK 다운로드
tPay_NFC_Demo.kt 
import com.tpay.nfc.TPayClient
import com.tpay.nfc.model.PaymentRequest

// 1. TPay NFC 결제 클라이언트 초기화
val client = TPayClient.Builder()
    .setMerchantId("TIT_TEST_1001")
    .setLogLevel(LogLevel.DEBUG)
    .build(context)

// 2. 승인 요청 객체 생성 (12,500원)
val request = PaymentRequest(
    amount = 12500,
    orderId = "ORD-20260615",
    trackData = "NFC_TOUCH_DATA..."
)

// 3. 무접촉 결제 대기 & 신호 감지
client.startPayment(request) { response ->
    if (response.isSuccess) {
        Log.d("SUCCESS", "승인번호: ${response.authCode}")
    }
}
